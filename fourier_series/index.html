Interactive Fourier Series

From a distance everything in the universe looks so much different from one another. But when we look closer we see that all these complex structures are just some combinations of simple atoms and molecules. In 1807 a French mathematician named Joseph Fourier took this idea one step further when he proved mathematically that any given complex function can be boiled down to a set of ordinary looking pure sine waves. He claimed that by adding only pure sine waves we can compose any complex shape or function. In other words any complex periodic function can be written as a sum of wieghted sine waves. This kind of expansion of a function is called the Fourier Series.

In this interactive demonstraion rotating circles has been used to represent each pure sinusoids. Amplitude generated by all the rotating circles are then summed together to draw the actual wave.
